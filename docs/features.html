E Export & Import

Seamless transfer from design tool to game engine

One-Click Export

Export single frames or batch export all frames in a page to a single ZIP archive containing structured JSON and optimized PNGs.

Scalable Assets

Export at 1x, 2x, 3x, or 4x resolution for crisp visuals on any display density without affecting layout calculations.

One-Click Import

Import exported JSON directly in Unity's editor window. Generates production-ready UGUI prefabs with all components configured.

Automatic Scene Setup

Auto-creates Canvas, EventSystem, and UIScreenManager if missing. Your scene is play-ready immediately after import.

Safe Re-import

Update designs and re-import without losing onClick handlers, custom MonoBehaviours, or inspector modifications.

Auto Re-import

Enable file watching to automatically re-import when export files change. Perfect for rapid iteration workflows.

L Layout & Responsive Design

Pixel-perfect layouts that adapt to any screen size

Auto-Layout Conversion

Figma auto-layouts become Unity HorizontalLayoutGroup or VerticalLayoutGroup with proper padding, spacing, and alignment.

Constraint System

Figma constraints (MIN, CENTER, MAX, STRETCH, SCALE) convert to Unity RectTransform anchors for responsive positioning.

Flex & Fill Support

Child flex/grow properties and FILL sizing modes translate to LayoutElement flexible width and height settings.

Space-Between Layout

Figma's space-between alignment creates invisible spacer GameObjects for proper distribution in Unity.

9-Slice Backgrounds

Automatic 9-slice detection from Figma layer structure. Creates Unity sliced sprites with correct border values.

Rich Text Support

TextMeshPro components with font family, size, weight, spacing, alignment, case, and text decoration preserved.

Layer Naming Conventions

Use these prefixes in Figma layer names to control how elements are imported:

Prefix Behavior Unity Components
BTN_ Button with baked composite image Image, Button
IMG_ Image sprite Image
ICON_ Icon sprite Image
CTN_ Container (no image, recurses children) RectTransform, LayoutGroup
9SLICE_ Scalable 9-slice background Image (Sliced)
TEXT Native text (auto-detected) TextMeshProUGUI

C Components & Variants

Design systems that work in both tools

Component to Prefab

Figma components become Unity prefabs. Instances use proper prefab linking for consistent updates across your project.

Variant State System

Figma component variants with properties like "State=Hover" become switchable states via FigmaVariantController at runtime.

Button State Auto-Wiring

Buttons with Default/Hover/Pressed/Disabled variants automatically respond to pointer events without any code.

Runtime State Control

Switch variant states programmatically with exact or partial key matching: controller.SetState("Hover")

N Navigation & Interactions

Prototype flows become working navigation

Prototype-Driven Navigation

Connect screens in Figma's prototype mode. Navigation links are automatically wired to Unity button onClick events.

Screen Manager

Built-in UIScreenManager singleton handles Show(), Back(), navigation history, and screen lifecycle management.

Screen Transitions

Configurable fade transitions between screens with customizable duration, hold time, and fade color.

Camera Effects

Four camera transition effects: Shake, ZoomPulse, DollyForward, and Drift for cinematic screen changes.

URL Actions

Figma prototype URL actions translate to Application.OpenURL calls, opening links in the system browser.

3D Environment Support

Auto-load matching 3D environment prefabs when screens change for hybrid 2D UI over 3D world setups.

A Entry Animations

Polished UI motion out of the box

Auto-Applied Animations

Entry animations automatically assigned based on layer prefix. Buttons pop in, images fade in, containers slide up.

Staggered Timing

Sibling elements animate with configurable stagger delay, creating a cascading reveal effect across the UI.

Spring Physics

PopIn animations feature spring overshoot and damping for organic, satisfying motion.

Custom Easing

Each animation supports custom AnimationCurve easing for precise control over motion timing.

D Developer Experience

Built for real production workflows

Zero-Code UI Setup

Designers create in Figma, developers import with one click. Working UI without writing layout code.

Iterative Updates

Re-export and re-import preserves developer customizations while updating visuals from the latest design.

Node Identity Tracking

Each node tracks its Figma ID via FigmaNodeKey. Renamed or moved elements stay linked across re-imports.

Orphan Cleanup

Nodes deleted in Figma are automatically removed from Unity prefabs, keeping your hierarchy clean.

Configurable PPU

Adjustable Pixels Per Unit setting for sprite sizing. Match your project's existing scale conventions.

Optional Extras

Generate 3D environment prefabs, screen transition prefabs, or video plane prefabs alongside screens.
