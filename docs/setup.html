Setup Guide

Get the Figma To Unity Pipeline running in minutes

Features Setup Guide Documentation
0

Requirements

Make sure you have these installed before starting:

U
Unity
2021.3 or later
F
Figma Desktop
Latest version
N
Node.js
16.x or later
P
Python
3.8+ (optional, for helper)
1

Install Unity Package

Copy the Assets/FigmaImporter folder into your Unity project:

YourUnityProject/
  Assets/
    FigmaImporter/ ← Copy this folder here
      Editor/
      Runtime/

Unity will automatically import the package. You should now see Tools > Figma > Import to Unity... in the menu bar.

TextMeshPro Required

If prompted, install TextMeshPro essentials via Window > TextMeshPro > Import TMP Essential Resources.

2

Build the Figma Plugin

Open a terminal in the FigmaPlugin folder and run:

cd FigmaPlugin
npm install
npm run build

This compiles code.ts into code.js which Figma needs to run the plugin.

Build Output

You should see: code.js 178.0kb and Done in XXms

3

Import Plugin to Figma

  1. Open Figma Desktop (not the web version)
  2. Go to Plugins > Development > Import plugin from manifest...
  3. Navigate to the FigmaPlugin folder
  4. Select manifest.json and click Open

The plugin is now installed. Access it via Plugins > Development > Figma to Unity UGUI Exporter.

Plugin Location

Don't move the FigmaPlugin folder after importing. Figma references the files at their original location.

4

Start Export Helper (Optional)

The helper script lets you save exports directly to a folder and use the Browse button. Run it in a terminal:

cd FigmaPlugin
python figma_export_helper.py

Leave it running while you work. You'll see:

==================================================
  Figma Export Helper
==================================================

Listening on: http://localhost:9876

Set your export path in the Figma plugin UI,
then click 'Save to Folder' to export directly.
Without the Helper

You can still use Download ZIP in the plugin and manually extract files to your Unity project.

5

Test the Pipeline

  1. In Figma: Create or open a design with a Frame
  2. Run the plugin: Plugins > Development > Figma to Unity UGUI Exporter
  3. Select a frame in your design
  4. Click "Export Selected Frame"
  5. Click "Download ZIP" (or "Save to Folder" if helper is running)
  6. Extract the ZIP to your Unity project's Assets folder
  7. In Unity: Go to Tools > Figma > Import to Unity...
  8. Click Browse and select the *_Screen.json file
  9. Click Import
Success!

Your Figma design is now a Unity prefab in Resources/Screens/. Press Play to see it!

Quick Reference

Once set up, your daily workflow is:

  1. Design in Figma using layer prefixes (BTN_, IMG_, CTN_, etc.)
  2. Run the plugin and click Export Selected Frame
  3. Save or download to your Unity project
  4. In Unity, use Tools > Figma > Import to Unity...
  5. Select the JSON file and click Import
