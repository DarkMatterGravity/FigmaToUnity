How To Use

Three simple steps from Figma to Unity

Features Setup Guide How To Use Documentation
1

Name Your Layers

Add prefixes to control how elements are imported:

BTN_ Clickable button IMG_ Static image ICON_ Icon image CTN_ Container (no image) 9SLICE_ Scalable background
Text is automatic

Text layers become TextMeshPro without any prefix needed.

2

Export from Figma

Select your frame, run the plugin, click Export Selected Frame, then Download ZIP.

Extract the ZIP to your Unity project's Assets folder.

3

Import to Unity

Go to Tools > Figma > Import to Unity...

Select the *_Screen.json file and click Import.

Your prefab is ready

Find it in Resources/Screens/ - press Play to see it!

+

Add Navigation (Optional)

In Figma's Prototype tab, connect buttons to other frames.

The importer wires these up automatically - clicks navigate between screens.
