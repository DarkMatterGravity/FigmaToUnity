Figma to Unity UGUI Exporter
Export Figma designs to Unity UI prefabs
Scale affects PNG resolution only. Layout and hierarchy are unchanged.
Files save directly here when using "Save to Folder" (requires helper running)

Layer Naming Conventions

CTN_
Container only (no image bake, recurses into children)
BTN_
Baked image + Unity Button component
IMG_
Baked composite image
ICON_
Baked icon image
9SLICE_
9-slice scalable background
TEXT
Native text (TextMeshPro, not baked)
Unprefixed groups under the root frame are automatically baked as composite images.
