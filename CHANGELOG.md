# Changelog

## [1.0.0] - 2026-06-12

### Added
- Initial release
- JSON file import from Figma plugin exports
- Figma REST API client (URL import coming soon)
- Auto-layout support (row/column → LayoutGroups)
- Constraint-based responsive anchoring
- Component and variant support with FigmaVariantController
- 9-slice image support
- TextMeshPro text styling
- Prototype navigation auto-wiring
- Entry animations (FadeIn, PopIn, SlideFrom variants)
- UIScreenManager for screen navigation
- Screen transitions (FadeTransition, CameraTransition)
- Auto-reimport on file changes
- Re-import safety (preserves onClick handlers and scripts)
